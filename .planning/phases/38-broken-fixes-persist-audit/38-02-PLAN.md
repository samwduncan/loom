<objective>
Audit all 5 Zustand stores for persist rehydration safety and verify the NaN fix.

Purpose: Shallow-merge rehydration is Zustand's default behavior with persist middleware. When `partialize` is used, only a subset of state is saved. On rehydration, shallow merge can clobber ephemeral fields with `undefined`, causing NaN displays, broken UI, or lost defaults. This audit ensures every persisted store has a proper deep merge function.

Output: All persisted stores have merge functions, comprehensive rehydration tests, verified NaN fix.
</objective>
