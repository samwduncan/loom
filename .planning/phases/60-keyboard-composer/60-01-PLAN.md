    - Export `async function initializeNativePlugins(): Promise<void>` that:
      1. Early-returns if `isInitialized` or `!IS_NATIVE`
      2. Sets `isInitialized = true`
      3. Sets `document.documentElement.setAttribute('data-native', '')` (for CSS conditional styling per D-08)
      4. Wraps everything in try/catch per D-04:
         - Dynamically imports `@capacitor/keyboard` via `await import('@capacitor/keyboard')` (D-03)
         - Stores the result in `KeyboardModule`
         - Destructures `{ Keyboard, KeyboardResize }` from the import
         - Calls `await Keyboard.setResizeMode({ mode: KeyboardResize.None })` (KEY-05, D-17)
         - Calls `await Keyboard.setAccessoryBarVisible({ isVisible: true })` (D-18)
      5. catch block: `console.warn('[native-plugins] Keyboard plugin failed to load:', err)` and sets `KeyboardModule = null`
      6. finally block: calls `resolveReady()` so the readiness promise resolves whether init succeeded or failed

    Named exports only (Constitution 2.2). No default export. Full JSDoc on the exported functions.
