Three changes:

**1. Add ambient gradient CSS to index.css (add BEFORE the @layer base block, after the spin keyframes):**

```css
/* Ambient gradient -- slowly cycling atmospheric background (ChatGPT-inspired) */
@property --ambient-hue {
  syntax: "<number>";
  initial-value: 0;
  inherits: false;
}

@keyframes ambient-cycle {
  0%   { --ambient-hue: 0; }     /* rose */
  25%  { --ambient-hue: 270; }   /* violet */
  50%  { --ambient-hue: 180; }   /* teal */
  75%  { --ambient-hue: 35; }    /* warm amber */
  100% { --ambient-hue: 360; }   /* rose (wrap) */
}

.ambient-gradient {
  position: fixed;
  inset: 0;
  pointer-events: none;
  z-index: 0;
  background: radial-gradient(
    ellipse at 50% 0%,
    hsl(var(--ambient-hue) 35% 55% / var(--fx-ambient-opacity, 0.04)) 0%,
    transparent 70%
  );
  animation: ambient-cycle var(--fx-ambient-speed, 45s) linear infinite;
}

@media (prefers-reduced-motion: reduce) {
  .ambient-gradient {
    animation: none;
    background: radial-gradient(
      ellipse at 50% 0%,
      hsl(0 35% 55% / 0.03) 0%,
      transparent 70%
    );
  }
}
```

Then add a `<div class="ambient-gradient"></div>` element. Since we should NOT modify React component files in this phase, instead use a body::before pseudo-element approach. Replace the above `.ambient-gradient` class with a `body::before` rule in @layer base:

```css
body::before {
  content: "";
  position: fixed;
  inset: 0;
  pointer-events: none;
  z-index: 0;
  background: radial-gradient(
    ellipse at 50% 0%,
    hsl(var(--ambient-hue) 35% 55% / var(--fx-ambient-opacity, 0.04)) 0%,
    transparent 70%
  );
  animation: ambient-cycle var(--fx-ambient-speed, 45s) linear infinite;
}
```

Also add the reduced-motion override for `body::before` inside the existing `@media (prefers-reduced-motion: reduce)` block.

IMPORTANT: Keep the `.ambient-gradient` class as well (as a reusable utility) -- it may be applied to other elements later. But the body::before is what provides the page-level atmospheric effect without React changes.
