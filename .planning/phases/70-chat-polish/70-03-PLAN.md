  <files>mobile/hooks/useSessions.ts</files>
  <action>
  The current useSessions hook groups sessions by project (ProjectWithSessions). Add temporal grouping as an alternative view of the same data.

  1. Add a TemporalGroup interface and grouping function:
     ```typescript
     export interface TemporalGroup {
       label: string;   // "Today", "Yesterday", "Previous 7 Days", "January 2026", etc.
       key: string;     // "today", "yesterday", "prev7", "2026-01", etc.
       sessions: SessionData[];
     }
     ```

  2. Create a `groupByTime(sessions: SessionData[]): TemporalGroup[]` function that:
     - Flattens all sessions from all projects into one array
     - Sorts by `updatedAt` descending (most recent first)
     - Buckets into temporal groups:
       - **Today**: updatedAt is today
       - **Yesterday**: updatedAt is yesterday
       - **Previous 7 Days**: updatedAt is 2-7 days ago
       - **Previous 30 Days**: updatedAt is 8-30 days ago
       - **Monthly**: For older sessions, group by "Month Year" (e.g., "January 2026")
     - Returns only non-empty groups, in chronological order (Today first)

  3. Add `temporalGroups` to the return type:
     ```typescript
     export interface UseSessionsReturn {
       // ... existing fields ...
       temporalGroups: TemporalGroup[];
     }
     ```

  4. Compute temporal groups from `allProjects` using `useMemo`:
     ```typescript
     const allSessions = useMemo(() =>
       allProjects.flatMap(p => p.sessions),
       [allProjects]
     );

     const temporalGroups = useMemo(() =>
       groupByTime(
         searchQuery.trim()
           ? allSessions.filter(s => s.title.toLowerCase().includes(searchQuery.toLowerCase()))
           : allSessions
       ),
       [allSessions, searchQuery]
     );
     ```

  5. Keep the existing `projects` field in the return value for backward compatibility, but the SessionList will now use `temporalGroups` instead.

  6. Add a `renameSession` callback that calls the backend API:
     ```typescript
     const renameSession = useCallback(async (projectName: string, sessionId: string, newTitle: string) => {
       try {
         await apiClient.apiFetch(
           `/api/projects/${encodeURIComponent(projectName)}/sessions/${encodeURIComponent(sessionId)}`,
           { method: 'PATCH', body: JSON.stringify({ title: newTitle }) }
         );
         await fetchSessions();
       } catch (e) {
         setError(e instanceof Error ? e.message : 'Failed to rename session');
       }
     }, [fetchSessions]);
     ```
     Add `renameSession` to the return object and interface.
