    4. **Add handleSlashSelect callback** (after handleMentionSelect):
       ```typescript
       const handleSlashSelect = useCallback(
         (cmd: SlashCommand | null) => {
           if (!cmd) return;

           // Clear the slash command text from input
           setInput('');

           // Execute the command action
           switch (cmd.id) {
             case 'clear': {
               const effectiveId = sessionId ?? streamSessionId;
               if (effectiveId) {
                 clearSession(effectiveId);
               }
               break;
             }
             case 'compact': {
               const effectiveId = sessionId ?? streamSessionId;
               if (effectiveId) {
                 wsClient.send({
                   type: 'claude-command',
                   command: '/compact',
                   options: { projectPath: projectName, sessionId: effectiveId },
                 });
               }
               break;
             }
             case 'help': {
               // Send /help as a regular message to the agent
               const effectiveId = sessionId ?? streamSessionId;
               if (effectiveId) {
                 wsClient.send({
                   type: 'claude-command',
                   command: '/help',
                   options: { projectPath: projectName, sessionId: effectiveId },
                 });
               }
               break;
             }
             case 'model': {
               // Send /model as a regular message to the agent
               const effectiveId = sessionId ?? streamSessionId;
               if (effectiveId) {
                 wsClient.send({
                   type: 'claude-command',
                   command: '/model',
                   options: { projectPath: projectName, sessionId: effectiveId },
                 });
               }
               break;
             }
           }

           closeSlashPicker();
           requestAnimationFrame(() => textareaRef.current?.focus());
         },
         [sessionId, streamSessionId, projectName, clearSession, closeSlashPicker],
       );
       ```

    5. **Update handleKeyDown** to handle slash picker (BEFORE the mention picker block, ~line 378):
       The slash picker takes priority when open. Add this block before the `if (mentionPickerOpen)` block:
       ```typescript
       if (slashPickerOpen) {
         if (e.key === 'ArrowUp') {
           e.preventDefault();
           slashMoveUp();
           return;
         }
         if (e.key === 'ArrowDown') {
           e.preventDefault();
           slashMoveDown();
           return;
         }
         if (e.key === 'Enter') {
           e.preventDefault();
           handleSlashSelect(slashSelectCurrent());
           return;
         }
         if (e.key === 'Tab') {
           e.preventDefault();
           handleSlashSelect(slashSelectCurrent());
           return;
         }
         if (e.key === 'Escape') {
           e.preventDefault();
           closeSlashPicker();
           return;
         }
       }
       ```
       Add slashPickerOpen, slashMoveUp, slashMoveDown, slashSelectCurrent, handleSlashSelect, closeSlashPicker to the handleKeyDown dependency array.

    6. **Update onChange handler** (in textarea onChange, after the detectAndOpen call ~line 485):
       ```typescript
       // Detect / slash command trigger
       slashDetectAndOpen(e.target.value, e.target.selectionStart ?? e.target.value.length);
       ```
       Note: Both detectors can run on every change. They won't conflict because:
       - Slash commands require `/` at position 0
       - Mentions require `@` preceded by whitespace or at position 0
       - A string starting with `/` won't trigger mention, and `@` mid-string won't trigger slash
