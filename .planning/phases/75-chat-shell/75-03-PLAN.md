       - `isActive === true` -> expanded (user sees thinking content arriving in real-time)
       - `isActive` transitions from true to false -> auto-collapse with Expand spring animation
       - User tap -> toggle expanded/collapsed

    2. **Expanded state:**
       - Subtle divider line at top: 1px height, `backgroundColor: theme.colors.border.subtle`, `marginVertical: theme.spacing.sm`
       - Thinking text: `theme.typography.body` (15px), `color: theme.colors.text.muted` (dimmed per D-07)

    3. **Collapsed state:**
       - Same divider line
       - Summary label: "Thought for Xs" where X is the real elapsed duration.
       - Use `theme.typography.small` (13px), `color: theme.colors.text.muted`
       - Pressable: tap anywhere on the collapsed summary to re-expand. 44px touch target minimum height.

    4. **AR fix #6: Duration tracking -- use real elapsed time, NOT character-count heuristic.**
       Track actual elapsed time via `useRef`:
       ```typescript
       const startTimeRef = useRef<number | null>(null);
       const [elapsedSeconds, setElapsedSeconds] = useState(0);

       useEffect(() => {
         if (isActive && !startTimeRef.current) {
           startTimeRef.current = Date.now();
         }
         if (!isActive && startTimeRef.current) {
           const elapsed = Math.max(1, Math.round((Date.now() - startTimeRef.current) / 1000));
           setElapsedSeconds(elapsed);
           // Don't reset startTimeRef -- one-shot capture
         }
       }, [isActive]);
       ```
       Display: `isActive ? "Thinking..." : `Thought for ${elapsedSeconds}s``
       This fixes the AR finding that character-count heuristic (`text.length / 200`) produces wrong values. Real elapsed time is accurate.

    5. **Animation per D-08 + UI-SPEC:**
       - Use Reanimated's `useAnimatedStyle` for height animation via the Expand spring config
       - Track `isExpanded` shared value (0 or 1)
       - Content height: measure with `onLayout`, store in ref. Animated height interpolates between 0 (collapsed -- only show summary line, ~24px) and measured content height (expanded)
       - Use `overflow: 'hidden'` on the animated container
       - On auto-collapse (isActive transitions false): `isExpanded.value = withSpring(0, theme.springs.expand)`
       - On tap: toggle `isExpanded.value = withSpring(isExpanded.value > 0.5 ? 0 : 1, theme.springs.expand)`

    6. **Haptic:** `Haptics.selectionAsync()` on tap to toggle.

    7. **Accessibility:** `accessibilityRole="button"` on the collapsed/expanded toggle area. `accessibilityLabel` includes "Thinking block" + expanded/collapsed state.
