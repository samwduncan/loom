**Sidebar Edge Zone Isolation (Forgejo #12):**

In `Sidebar.tsx`, modify the swipe-to-close touch handlers to only activate from the right edge of the sidebar panel (the area closest to the main content). Interior horizontal swipes must pass through to child gesture handlers (useSwipeToDelete on SessionItem).

1. In `handleTouchStart` (line 57-65), after capturing `startX`, calculate whether the touch is in the edge zone:
```typescript
const handleTouchStart = useCallback((e: React.TouchEvent) => {
  const touch = e.touches[0];
  if (!touch) return;
  touchRef.current.startX = touch.clientX;
  touchRef.current.swiping = false;

  // Gesture zone isolation: only activate sidebar-close from the right edge
  // (last 30px of sidebar panel width, closest to main content area).
  // Interior swipes fall through to child handlers (swipe-to-delete).
  const EDGE_ZONE_WIDTH = 30;
  const sidebarWidth = sidebarPanelRef.current?.offsetWidth ?? 280;
  touchRef.current.isEdgeSwipe = touch.clientX > (sidebarWidth - EDGE_ZONE_WIDTH);

  if (sidebarPanelRef.current && touchRef.current.isEdgeSwipe) {
    sidebarPanelRef.current.style.transition = 'none';
  }
}, []);
```

2. Update the `touchRef` type to include the new `isEdgeSwipe` field:
```typescript
const touchRef = useRef({ startX: 0, swiping: false, isEdgeSwipe: false });
```

**Delete Button Z-Ordering Fix (Forgejo #17):**

In `SessionItem.tsx`, restructure the delete button to be a sibling OUTSIDE the gesture-bound div instead of behind it. The current structure has the delete button behind the content div, but the content div's gesture handler prevents taps from reaching it even when the content is translated left.

Replace the current structure (lines 156-274) with:
```tsx
<div className="relative overflow-hidden">
  {/* Session item content (translates via offset on mobile swipe) */}
  <div
    {...(isMobile ? bind() : {})}
    style={{
      transform: isMobile && offset !== 0 ? `translateX(${offset}px)` : undefined,
      transition: active ? 'none' : 'transform var(--duration-normal) var(--ease-out)',
      touchAction: isMobile ? 'pan-y' : 'auto',
    }}
  >
    {/* ... existing role="option" inner div with all session content unchanged ... */}
  </div>
  {/* Delete button -- OUTSIDE gesture binding, absolutely positioned at right edge */}
  {/* Rendered above gesture layer with z-10 so taps are not intercepted */}
  {isMobile && (offset < 0 || revealed) && (
    <button
      onClick={handleSwipeDelete}
      className={cn(
        'absolute right-0 top-0 bottom-0 w-[80px] z-10',
        'flex flex-col items-center justify-center gap-0.5',
        'text-white font-medium',
        'min-h-[44px]',
      )}
      style={{
        background: 'var(--status-error)',
        pointerEvents: 'auto',
      }}

  )}
</div>
```
