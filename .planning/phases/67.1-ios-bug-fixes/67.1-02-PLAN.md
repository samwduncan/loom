**Sidebar Edge Zone Isolation (Forgejo #12):**

In `Sidebar.tsx`, modify the swipe-to-close touch handlers to only activate from the right edge of the sidebar panel (the area closest to the main content). Interior horizontal swipes must pass through to child gesture handlers (useSwipeToDelete on SessionItem).

1. In `handleTouchStart` (line 57-65), use `getBoundingClientRect()` for reliable edge zone detection (not raw clientX which is fragile when sidebar offset changes):
```typescript
const handleTouchStart = useCallback((e: React.TouchEvent) => {
  const touch = e.touches[0];
  if (!touch) return;
  touchRef.current.startX = touch.clientX;
  touchRef.current.swiping = false;

  // Gesture zone isolation: only activate sidebar-close from the right edge
  // (last 30px of sidebar panel width, closest to main content area).
  // Interior swipes fall through to child handlers (swipe-to-delete).
  // Use getBoundingClientRect for reliable position regardless of sidebar transform.
  const EDGE_ZONE_WIDTH = 30;
  const rect = sidebarPanelRef.current?.getBoundingClientRect();
  touchRef.current.isEdgeSwipe = rect
    ? touch.clientX > (rect.right - EDGE_ZONE_WIDTH)
    : false;

  if (sidebarPanelRef.current && touchRef.current.isEdgeSwipe) {
    sidebarPanelRef.current.style.transition = 'none';
  }
}, []);
```

2. Update the `touchRef` type to include the new `isEdgeSwipe` field:
```typescript
const touchRef = useRef({ startX: 0, swiping: false, isEdgeSwipe: false });
```

3. In `handleTouchMove` (line 67-75), add early return if not an edge swipe:
```typescript
const handleTouchMove = useCallback((e: React.TouchEvent) => {
```
