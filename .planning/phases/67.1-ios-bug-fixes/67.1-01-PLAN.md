**Haptics Diagnostic Logging (Forgejo #16):**

In the haptics plugin try/catch block (lines 144-149), add a console.log BEFORE and AFTER setHapticsModule to help diagnose if haptics fail on device:

```typescript
try {
  const hapticsMod = await import('@capacitor/haptics');
  console.log('[native-plugins] Haptics module loaded successfully:', !!hapticsMod?.Haptics);
  setHapticsModule(hapticsMod);
} catch (err: unknown) {
  console.warn('[native-plugins] Haptics plugin failed to load:', err);
  setHapticsModule(null);
}
```

This diagnostic log will be visible in Xcode console during device testing and helps debug #16 if the dep move alone doesn't fix haptics.
