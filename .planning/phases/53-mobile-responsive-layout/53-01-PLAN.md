    **MOBILE-02 Touch Target Fixes:**

    Use the `md:` breakpoint pattern: apply larger touch targets on mobile (default), restore desktop sizes with `md:` prefix. The project convention is mobile-first with `md:` (768px+) for desktop overrides.

    **A. ChatView header buttons** (ChatView.tsx lines ~210-270):
    The three header control buttons (thinking toggle, search toggle, export toggle) use `p-1.5` with `size-4` icons = ~28px. On mobile, these must be 44x44px.
    - Add `min-h-[44px] min-w-[44px] md:min-h-0 md:min-w-0` to each of the three header buttons
    - Increase padding on mobile: change `p-1.5` to `p-3 md:p-1.5` on each button
    - Export dropdown items: change `py-1.5` to `py-3 md:py-1.5` on both export buttons

    **B. Tool chips** (tool-chip.css):
    The `.tool-chip` button has `padding: 4px 10px` = ~26px height. Add a mobile media query:
    ```css
    @media (max-width: 767px) {
      .tool-chip {
        min-height: 44px;
        padding: 10px 14px;
      }
    }
    ```

    **C. Session items** (SessionItem.tsx):
    The session item div uses `px-3 py-2` = ~36px. Add `min-h-[44px]` to the session item root div className. This is mobile-only via: `min-h-[44px] md:min-h-0`.

    **D. Sidebar buttons** (Sidebar.tsx):
    - Mobile hamburger button (line ~63): currently `p-2`. Change to `p-3` to get 44px (20px icon + 24px padding = 44px).
    - Sidebar close button (line ~117): currently `p-1`. Change to `p-3 md:p-1` for mobile touch target.
    - Settings button in footer (line ~137): currently `p-2`. Change to `p-3 md:p-2` for mobile touch target.

    **E. CodeBlock copy button** (CodeBlock.tsx):
    The copy button in the header bar has no padding/min-height. Add `min-h-[44px] md:min-h-0 px-2 md:px-0` to the copy button's className.

    **F. Composer send/stop buttons** (ChatComposer.tsx):
    Send and stop buttons are `h-8 w-8` (32px). Change to `h-11 w-11 md:h-8 md:w-8` for 44px on mobile.

    **G. Composer mention chip remove buttons** (composer.css):
    The `.mention-chip-remove` is tiny. Add mobile sizing:
    ```css
    @media (max-width: 767px) {
      .mention-chip-remove {
        min-width: 28px;
        min-height: 28px;
        display: flex;
        align-items: center;
        justify-content: center;
      }

        min-height: 36px;
        padding: 6px 12px;
      }
    }
    ```
