Build the tool call grouping system for historical (finalized) messages: a pure grouping algorithm, the ToolCallGroup component with CSS Grid animation, and AssistantMessage integration.

Purpose: Consecutive tool calls (often 5-20 in a row) overwhelm the chat view. Grouping them into collapsible accordions keeps the interface scannable while preserving access to individual tool details.

Output: groupToolCalls utility, ToolCallGroup component + CSS, updated AssistantMessage rendering grouped tools instead of individual markers.
