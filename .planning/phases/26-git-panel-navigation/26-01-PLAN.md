       - Returns `{ data: GitStatusData | null; loading: boolean; error: string | null; refetch: () => void }`
       - Flattens backend's `{modified, added, deleted, untracked}` arrays into a single `GitFileChange[]`

       - Uses AbortController for cleanup
       - Initial fetch on mount via useEffect

    3. Create `src/src/hooks/useGitBranches.ts`:
       - Same FetchState pattern
       - Fetches GET `/api/git/branches?project=...`
       - Returns `{ branches: GitBranch[] | null; loading: boolean; error: string | null; refetch: () => void }`

    4. Create `src/src/hooks/useGitCommits.ts`:
       - Same FetchState pattern
       - Fetches GET `/api/git/commits?project=...`
       - Returns `{ commits: GitCommit[] | null; loading: boolean; error: string | null; refetch: () => void }`

    5. Create `src/src/hooks/useGitRemoteStatus.ts`:
       - Same FetchState pattern
       - Fetches GET `/api/git/remote-status?project=...`
       - Returns `{ remoteStatus: GitRemoteStatus | null; loading: boolean; error: string | null; refetch: () => void }`

    6. Create `src/src/hooks/useGitOperations.ts`:
       - Takes `projectName: string`
       - Returns object of async functions (NOT hooks -- these are imperative actions):
         - `commit(message: string, files: string[]): Promise<void>` -- POST /api/git/commit
         - `push(): Promise<void>` -- POST /api/git/push
         - `pull(): Promise<void>` -- POST /api/git/pull
         - `fetch(): Promise<void>` -- POST /api/git/fetch
         - `checkout(branch: string): Promise<void>` -- POST /api/git/checkout
         - `createBranch(name: string): Promise<void>` -- POST /api/git/create-branch
         - `discard(file: string): Promise<void>` -- POST /api/git/discard
         - `deleteUntracked(file: string): Promise<void>` -- POST /api/git/delete-untracked
         - `generateCommitMessage(): Promise<string>` -- POST /api/git/generate-commit-message
       - Each function takes project from closure, sends JSON body with `{ project: projectName, ...params }`
       - Use `useCallback` to memoize the returned object based on projectName

    7. Create `src/src/hooks/useGitStatus.test.ts`:
