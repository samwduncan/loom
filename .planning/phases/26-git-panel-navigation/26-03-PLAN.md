    1. Create `src/src/components/git/BranchSelector.tsx`:
       - Props: `{ currentBranch: string; projectName: string; onBranchChange: () => void }`
       - Uses `useGitBranches(projectName)` for branch list
       - Uses `useGitOperations(projectName)` for checkout and createBranch
       - Renders current branch name as a button/trigger
       - On click: shows dropdown list of branches (use a simple popover or shadcn Select)
         - Each branch item shows name, current branch has a checkmark
         - Clicking a branch calls checkout(branchName), then refetches branches and calls onBranchChange

       - Loading state during checkout/create shows spinner on the button

       - Props: `{ branch: string; projectName: string; onRefresh: () => void }`
       - Uses `useGitRemoteStatus(projectName)` for ahead/behind
       - Uses `useGitOperations(projectName)` for push, pull, fetch
       - Layout: horizontal bar with:
         a. `<BranchSelector>` on the left
         b. Remote status badge in center: shows "ahead N / behind M" if hasRemote && hasUpstream
            - If !hasRemote: show nothing (no push/pull/fetch either)
            - If hasRemote but !hasUpstream: show "No upstream" with option to publish
         c. Push (ArrowUp), Pull (ArrowDown), Fetch (RefreshCw) icon buttons on right
            - Each button: disabled when loading, shows spinner during operation
            - Each button: tooltip via title attribute
            - onClick: call operation, await, toast.success/error, refetch remote status, call onRefresh
            - Conditionally render based on remoteStatus.hasRemote
       - Local state for per-button loading: `{ pushing: boolean; pulling: boolean; fetching: boolean }`

    3. Wire GitPanelHeader into GitPanel.tsx:
       - Render `<GitPanelHeader>` above the sub-tabs when data is loaded
       - Pass branch from status data, projectName, and refetch as onRefresh

    4. Update git-panel.css:
       - `.git-header` flex row, items-center, gap, padding
       - `.git-remote-badge` small badge styling with ahead/behind numbers
       - `.git-action-btn` icon button with hover/active states, disabled opacity

    5. Create `src/src/components/git/GitPanelHeader.test.tsx`:
