Three changes in order:

**1. Rewrite public/manifest.json** with correct Loom branding and dark theme:
```json
{
  "name": "Loom",
  "short_name": "Loom",
  "description": "AI agent workspace — make agent work visible, beautiful, and controllable",
  "start_url": "/",
  "scope": "/",
  "display": "standalone",
  "orientation": "any",
  "theme_color": "#2b2521",
  "background_color": "#2b2521",
  "icons": [
    {
      "src": "/icons/icon-192x192.png",
      "sizes": "192x192",
      "type": "image/png",
      "purpose": "any maskable"
    },
    {
      "src": "/icons/icon-512x512.png",
      "sizes": "512x512",
      "type": "image/png",
      "purpose": "any maskable"
    }
  ]
}
```

Notes on choices:
- theme_color `#2b2521` is the closest hex to `oklch(0.20 0.010 32)` (--surface-base token) — dark warm brown that matches the app chrome
- background_color matches theme_color for a seamless splash screen
- orientation: any (per CONTEXT.md — works portrait and landscape)
- display: standalone (shows status bar; not fullscreen)
- Only 192 and 512 icons needed — browsers pick the right one; more sizes add no value
- purpose: "any maskable" allows both adaptive icons (maskable) and standard usage (any)

**2. Update src/index.html** — add manifest link and meta tags inside `<head>`, after the existing `<link rel="icon">` line:
```html
    <link rel="manifest" href="/manifest.json" />
    <meta name="theme-color" content="#2b2521" />
    <meta name="apple-mobile-web-app-capable" content="yes" />
    <meta name="apple-mobile-web-app-status-bar-style" content="black-translucent" />
    <meta name="apple-mobile-web-app-title" content="Loom" />
    <link rel="apple-touch-icon" href="/icons/icon-192x192.png" />
```

Preserve ALL existing tags (charset, icon, viewport, title). Do NOT add any service worker registration script.

**3. Delete public/sw.js** — this file is NOT referenced anywhere in the codebase (confirmed by grep). Its presence could confuse browsers into thinking a SW is available. Per STATE.md decision: "No service worker in v2.0 -- manifest-only PWA to avoid stale content trap." Delete it with `rm public/sw.js` via Bash.
