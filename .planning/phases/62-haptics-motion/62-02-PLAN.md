<objective>
Tune spring animations for ProMotion and enable 120Hz rendering via Info.plist.

Purpose: MOTION-01/02/03 require that CSS animations run at 120Hz on ProMotion displays and spring configs feel snappier. These are physics-based curves -- more frames render the same curve more smoothly, no separate 120Hz variants needed (D-20).
