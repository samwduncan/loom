Purpose: NATIVE-03 requires haptic feedback on message send, tool completion, and error states. D-13 through D-16 specify the exact integration points and haptic types.     const prevStatusRef = useRef(toolCall.status);

    ```typescript
      if (toolCall.status === 'resolved') {

        hapticNotification('Error'); // D-15: haptic on tool error

    ```
