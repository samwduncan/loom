## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px (`--space-1`) | Icon gaps, inline padding |
| sm | 8px (`--space-2`) | Compact element spacing, context menu item padding |
| md | 16px (`--space-4`) | Default element spacing, context menu content padding |
| lg | 24px (`--space-6`) | Section padding |
| xl | 32px (`--space-8`) | Layout gaps |
| 2xl | 48px (`--space-12`) | Major section breaks |
| 3xl | 64px (`--space-16`) | Page-level spacing |

Exceptions:
- **44px**: Mobile touch targets (min-height on interactive elements, per V2_CONSTITUTION Section 13)
- **80px**: Swipe-to-delete reveal zone width (D-09). The delete button behind the swiped session item occupies 80px horizontal.
- **60px**: Pull-to-refresh activation threshold (D-12). Spinner container height before release trigger.

Source: `src/src/styles/tokens.css` lines 117-130, CONTEXT.md D-09/D-12

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `oklch(0.20 0.010 32)` `--surface-base` | Background, scroll container surfaces |
| Secondary (30%) | `oklch(0.23 0.008 32)` `--surface-raised` | Session list items, context menu content background (via `bg-popover`) |
| Accent (10%) | `oklch(0.63 0.14 20)` `--accent-primary` | Context menu focus highlight (`bg-accent`), pull-to-refresh spinner stroke |
| Destructive | `oklch(0.65 0.18 25)` `--status-error` | Swipe-to-delete button background, destructive context menu items, action sheet destructive option |

Accent reserved for:
- Context menu item focus/hover highlight (`focus:bg-accent`)
- Pull-to-refresh spinner stroke color
- Active gesture indicator (swipe threshold reached)

Destructive reserved for:
- Swipe-to-delete revealed button (red background, white text)
- Context menu "Delete" item (`variant="destructive"`)
- Native action sheet destructive button style (`ActionSheetButtonStyle.Destructive`)

## Interaction Contract

This phase is interaction-heavy. The following contracts define gesture physics, timing, and visual feedback.

### Swipe-to-Delete (GESTURE-01)

| Property | Value |
|----------|-------|
| Direction | Left only (negative X) |
| Activation threshold | 80px horizontal displacement OR velocity > 0.5 |
| Axis lock | `axis: 'x'`, `threshold: [10, Infinity]` |
| Delete button width | 80px |
| Delete button color | `--status-error` background, white text |
| Delete button icon | `Trash2` from lucide-react, 18px |
| Snap-back animation | `transition: transform var(--duration-normal) var(--ease-out)` (200ms ease-out) |
| Reveal animation | `transition: transform var(--duration-fast) var(--ease-out)` (100ms ease-out) |
| Haptic on reveal | `hapticImpact('Light')` |
| Haptic on confirm | `hapticNotification('Warning')` |
| Mobile only | `< 768px` (md breakpoint) |
| Touch action CSS | `touch-action: pan-y` on swipeable element |
| Accessibility | Delete key / Backspace on focused session item |

### Pull-to-Refresh (GESTURE-02)

| Property | Value |
|----------|-------|
| Direction | Down only (positive Y), at scrollTop === 0 |
| Activation threshold | 60px vertical displacement |
| Axis lock | `axis: 'y'` |
| Spinner size | 24px diameter circle |
| Spinner stroke | 2px, `--accent-primary` color |
| Spinner animation | CSS `rotate` keyframe, 800ms linear infinite |
| Pull indicator | Proportional opacity (0 at 0px, 1 at 60px) during drag |
| Overscroll CSS | `overscroll-behavior-y: contain` on session list container |
| Snap-back animation | `transition: transform var(--duration-normal) var(--ease-spring-snappy)` (200ms spring) |
| Haptic on complete | `hapticNotification('Success')` |
| Error state | Toast notification via Sonner on fetch failure |
| Mobile only | `< 768px` (md breakpoint) |
| Touch action CSS | `touch-action: pan-x` on pull container (allow horizontal but capture vertical at top) |

### Long-Press Context Menu (GESTURE-03, GESTURE-06)

| Property | Value |
|----------|-------|
| Trigger | Radix ContextMenu built-in long-press (touch devices), right-click (desktop) |
| Long-press duration | Radix default (test on device; override via controlled state + useDrag delay:500 if needed) |
| Haptic on open | `hapticImpact('Medium')` |
| Menu positioning | Radix auto-positioning (avoids viewport edges) |
| Menu appearance | `animate-in fade-in-0 zoom-in-95` (existing shadcn animation) |
| Menu disappearance | `animate-out fade-out-0 zoom-out-95` (existing shadcn animation) |
| Text selection prevention | `-webkit-user-select: none` on ContextMenuTrigger wrapper (mobile only) |
| Z-index | `var(--z-dropdown)` (20) |
| Min width | 8rem (128px) per existing shadcn ContextMenuContent |
| Item height | `py-1.5` (6px top + 6px bottom = 12px padding + 14px text = ~26px visible, meets touch target via Radix focus management) |

### Session Context Menu Items (GESTURE-03)

| Item | Icon | Variant | Haptic |
|------|------|---------|--------|
| Copy ID | `Copy` 14px | default | `hapticSelection()` on select |
| Rename | `Pencil` 14px | default | none |
| Pin / Unpin | `Pin` 14px | default | `hapticImpact('Light')` |
| Export | `Download` 14px | default | none |
| --- (separator) | -- | -- | -- |
| Delete | `Trash2` 14px | destructive | triggers action sheet confirmation |

### Message Context Menu Items (GESTURE-06)

| Item | Icon | Variant | Haptic |
|------|------|---------|--------|
| Copy Text | `Copy` 14px | default | `hapticSelection()` on select |
| Retry | `RotateCcw` 14px | default | none |
| --- (separator) | -- | -- | -- |
| Share | `Share` 14px | default | `hapticImpact('Light')` |

### Haptic Feedback Grammar (GESTURE-05)

| Event | Function | Style | Meaning |
|-------|----------|-------|---------|
| Session select (tap) | `hapticSelection()` | selection changed | Item acknowledged |
| Sidebar toggle | `hapticImpact('Light')` | light thud | Panel state change |
| Swipe-to-delete reveal | `hapticImpact('Light')` | light thud | Action revealed |
| Delete confirmed | `hapticNotification('Warning')` | warning pulse | Destructive action |
| Context menu open | `hapticImpact('Medium')` | medium thud | Affordance |
| Pull-to-refresh complete | `hapticNotification('Success')` | success pulse | Data loaded |
| Share triggered | `hapticImpact('Light')` | light thud | Acknowledgment |

All haptic events silently no-op when: (1) not native, (2) haptics module not loaded, (3) `prefers-reduced-motion` is active. Existing architecture in `haptics.ts` handles all three gates.

## Motion Contract

| Animation | Duration | Easing | Reduced Motion |
|-----------|----------|--------|----------------|
| Swipe reveal (active drag) | Follows finger (no duration) | Linear (1:1 tracking) | Instant snap to 0 or -80px |
| Swipe snap-back | 200ms `--duration-normal` | `--ease-out` | Instant |
| Swipe auto-reveal (velocity) | 100ms `--duration-fast` | `--ease-out` | Instant |
| Pull-to-refresh spring | 200ms | `--ease-spring-snappy` | Instant |
| Pull-to-refresh spinner | 800ms | linear (rotation) | Static indicator (no spin) |
| Context menu open | ~150ms | `fade-in-0 zoom-in-95` (shadcn) | Instant (no animation) |
| Context menu close | ~150ms | `fade-out-0 zoom-out-95` (shadcn) | Instant (no animation) |
| Delete button fade | 150ms | `--ease-out` | Instant |

All animations use CSS transitions (per CSS-first motion decision from v2.1). No JS animation frames for gesture feedback -- refs track position, CSS transitions handle snap.

Source: `src/src/styles/tokens.css` motion tokens, RESEARCH.md anti-patterns

## Registry Safety

| Registry | Blocks Used | Safety Gate |
|----------|-------------|-------------|
| shadcn official | context-menu (existing), alert-dialog (existing) | not required |

No third-party registries. No new shadcn components needed -- existing `context-menu.tsx` and `alert-dialog.tsx` primitives cover all menu and confirmation UI. The new `PullToRefreshSpinner` is a custom component (not from registry).

New npm dependencies:
- `@use-gesture/react@^10.3.1` -- production dependency, gesture hooks
- `@capacitor/app@^7.1.2` -- devDependency, app lifecycle
- `@capacitor/share@^7.0.4` -- devDependency, native share
- `@capacitor/action-sheet@^7.0.4` -- devDependency, native action sheet
- `@capacitor/clipboard@^7.0.4` -- devDependency, native clipboard

All Capacitor plugins are 7.x versions for compatibility with `@capacitor/core@^7.6.1`. NEVER install 8.x.

Source: RESEARCH.md Standard Stack, CONTEXT.md D-29/D-30
