Create `src/src/lib/platform.ts` with these exact exports (per D-01, D-02, D-05 through D-10):

```typescript
export const IS_NATIVE: boolean
export const API_BASE: string
export const WS_BASE: string
export function resolveApiUrl(path: string): string
export function resolveWsUrl(path: string, token: string): string
export function resolveShellWsUrl(token: string): string
export async function fetchAnon(path: string, options?: RequestInit): Promise<Response>
```

Implementation details:
- `IS_NATIVE`: `typeof window !== 'undefined' && typeof (window as Record<string, unknown>).Capacitor !== 'undefined'` (per D-01)
- `API_BASE`: `IS_NATIVE ? (import.meta.env.VITE_API_URL as string | undefined) ?? 'http://100.86.4.57:5555' : ''` (per D-08, D-09)
- `WS_BASE`: `IS_NATIVE ? API_BASE.replace(/^http/, 'ws') : ''` (derived from API_BASE)
- `resolveApiUrl(path)`: returns `${API_BASE}${path}` (per D-05)
- `resolveWsUrl(path, token)`: if IS_NATIVE, returns `${WS_BASE}${path}?token=${token}`; else builds from `window.location.protocol` (wss: if https:, else ws:) + `window.location.host` + path + `?token=${token}` (per D-06)
- `resolveShellWsUrl(token)`: returns `resolveWsUrl('/shell', token)` (per D-07)
- `fetchAnon(path, options)`: calls `fetch(resolveApiUrl(path), { ...options, headers: { 'Content-Type': 'application/json', ...(options.headers as Record<string, string> | undefined) } })` (per D-10)
- All named exports, no default export (Constitution 2.2)
- JSDoc comments on each export explaining web vs native behavior

For testing native mode: Since IS_NATIVE, API_BASE, WS_BASE are module-scoped constants evaluated at import time, the tests for native mode need to use `vi.stubGlobal` to set `window.Capacitor` BEFORE importing the module, then use `vi.resetModules()` + dynamic `import()` to get fresh evaluation. Structure the test file with two describe blocks:
1. `describe('web mode (default)')` -- tests against the statically imported module
2. `describe('native mode (mocked)')` -- uses `vi.stubGlobal('Capacitor', { isNativePlatform: () => true })` + `vi.resetModules()` + dynamic import

The test file should use Vitest (`import { describe, it, expect, vi } from 'vitest'`), mock `globalThis.fetch` for fetchAnon tests.
