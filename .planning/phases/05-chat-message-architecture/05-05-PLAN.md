     a. Claude `content_block_delta` handler (~line 256)
     b. Claude `claude-output` handler (~line 464)

     ```typescript
     if (!streamTimerRef.current) {
       streamTimerRef.current = requestAnimationFrame(() => {
         const chunk = streamBufferRef.current;
         streamBufferRef.current = '';
         streamTimerRef.current = null;
         appendStreamingChunk(setChatMessages, chunk, false); // or true for newline variants
     ```

   - The `as unknown as number` cast is needed because rAF returns a number but the ref type is `number | null`

     Option B: Thread `isStreaming` prop through Markdown to CodeBlock via react-markdown's components customization.

   - This ensures code blocks in streaming messages show raw text fallback, then switch to Shiki highlighting once streaming completes and `isStreaming` flips to false.
