<objective>
Build the compact tool action card system with type-based tinting, expandable details, error states, and automatic grouping of consecutive tool calls.

Purpose: CHAT-04 through CHAT-07 transform tool call rendering from the current two-tier OneLineDisplay/CollapsibleDisplay system into a unified compact card design. Cards show tool name + key argument on a single line with a status icon, expand inline to reveal full details, and automatically group when 3+ consecutive calls appear. This dramatically reduces visual noise in agentic sessions.

Output: ToolActionCard component, ToolCallGroup component, groupConsecutiveToolCalls utility, updated ToolRenderer and MessageComponent.
</objective>

   - Export `groupConsecutiveToolCalls(messages: ChatMessage[]): (ChatMessage | ChatMessage[])[]`
   - Pure function (no hooks, no React dependency) -- takes a flat array of ChatMessage and returns a mixed array where consecutive runs of 3+ `isToolUse` messages (excluding `isSubagentContainer`) are grouped into sub-arrays

     b. Accumulate consecutive `isToolUse && !isSubagentContainer` messages
