         Derive a boolean `isMobile` using `useSyncExternalStore` with `window.matchMedia('(max-width: 767px)')`:
         ```
         const mql = window.matchMedia('(max-width: 767px)');
         const isMobile = useSyncExternalStore(
           (cb) => { mql.addEventListener('change', cb); return () => mql.removeEventListener('change', cb); },
           () => mql.matches,
           () => false // SSR fallback
         );
         const activeTab = isMobile ? 'chat' : rawActiveTab;
         ```
         This avoids the matchMedia useEffect flash — the override is synchronous in the render path. The store is NOT mutated; we just override locally for rendering. No `useEffect` needed.
