1. Create `useRecentCommands.ts`:
   - localStorage key: `'loom-recent-commands'`
   - `MAX_RECENT = 10`
   - Interface: `RecentEntry { id: string; label: string; group: string; timestamp: number }`
   - Export `useRecentCommands()` hook returning `{ recents: RecentEntry[]; addRecent: (entry: Omit<RecentEntry, 'timestamp'>) => void; clearRecents: () => void }`
   - `addRecent`: reads from localStorage, deduplicates by id, prepends new entry with `Date.now()` timestamp, slices to MAX_RECENT, writes back
   - `recents`: reads from localStorage on mount (useState + useEffect), re-reads after addRecent
   - Use `useSyncExternalStore` pattern if simpler, but useState+useEffect is fine for this

2. Create `useCommandSearch.ts`:
   - Accepts `search: string` parameter
   - Creates three Fuse instances via useMemo (rebuilt when source data changes):
     - Sessions fuse: keys `['title']`, threshold 0.4, source from `useTimelineStore((s) => s.sessions)`
     - Files fuse: keys `['path', 'name']`, threshold 0.3, source from state (fetched once on palette open)
     - Commands fuse: built-in static commands (Navigation, Actions) + slash commands from API
   - Returns `{ sessionResults: Session[]; fileResults: FileEntry[]; commandResults: CommandEntry[]; isLoading: boolean }`
   - When search is empty, returns empty arrays (caller shows recent commands instead)
   - Debounce NOT needed (fuse.js is fast enough for <1000 items)
   - File fetching: `apiFetch<FileEntry[]>('/api/projects/${projectName}/files')` — fetch once when palette opens (controlled by parent via `enabled` boolean). If fetch fails, fileResults stays empty (graceful degradation).
   - Limit all fuse.search calls to `{ limit: 15 }`
   - Define interfaces: `FileEntry { path: string; name: string; type: 'file' | 'directory' }`, `CommandEntry { id: string; label: string; group: string; shortcut?: string; action: () => void }`
   - Export `CommandEntry` and `FileEntry` types for use by group components in Plan 02

3. Create `useRecentCommands.test.ts`:

   - Test: addRecent deduplicates by id (moves to top)
