1. **NavigationGroup.tsx** (CMD-06):
   - 5 items: Switch to Chat (shortcut: `⌘1`), Switch to Files (`⌘2`), Switch to Shell (`⌘3`), Switch to Git (`⌘4`), Open Settings
   - Icons: MessageSquare, FolderTree, Terminal, GitBranch, Settings (from lucide-react)
   - onSelect: call `useUIStore.getState().setActiveTab(tab)` then `onClose()`
   - Open Settings: call `useUIStore.getState().openModal({ type: 'settings' })` then `onClose()`
   - Each item tracks recent via `addRecent({ id: 'nav-chat', label: 'Switch to Chat', group: 'Navigation' })`

2. **SessionGroup.tsx** (CMD-07, CMD-08):
   - Props: `sessions: Session[]` (filtered by useCommandSearch), `onClose: () => void`, `addRecent`
   - Only renders when sessions array is non-empty
   - Each item: label = session.title, icon = MessageSquare, no shortcut
   - onSelect: `useUIStore.getState().setActiveTab('chat')` then `useTimelineStore.getState().setActiveSession(session.id)` then use `navigate('/chat/' + session.id)` (import from react-router-dom) then `onClose()`
   - Note: SessionGroup needs react-router-dom's `useNavigate` for navigation

3. **FileGroup.tsx** (CMD-10):
   - Props: `files: FileEntry[]` (filtered by useCommandSearch), `onClose: () => void`, `addRecent`
   - Only renders when files array is non-empty
   - Each item: label = file.name, sublabel = file.path (render path in text-muted after name), icon = FileText
   - onSelect: `useUIStore.getState().setActiveTab('files')` then `useFileStore.getState().openFile(file.path)` then `onClose()`
   - Note: file store `openFile` may not exist yet — use a stub: `console.warn('TODO: openFile not implemented', path)` and just switch to Files tab. Phase 23 will wire this.

4. **ActionGroup.tsx** (CMD-09):
   - 3 items: New Session (icon: Plus), Toggle Thinking Visibility (icon: Eye), Toggle Sidebar (icon: PanelLeft)
   - New Session: navigate to `/chat` (no session ID) which triggers new session creation flow, then `onClose()`
   - Toggle Thinking: `useUIStore.getState().toggleThinking()` then `onClose()`
   - Toggle Sidebar: `useUIStore.getState().toggleSidebar()` then `onClose()`

5. **CommandGroup.tsx** (CMD-11):
   - Props: `search: string`, `onClose: () => void`, `addRecent`
   - Fetches slash commands from `POST /api/commands/list` via apiFetch when search starts with '/' or when search matches command names
   - Use `useState` + `useEffect` to fetch once on mount, cache in ref
   - Interface: `SlashCommand { name: string; description: string; namespace: string }`
   - Each item: label = command.name, sublabel = command.description, icon = Slash (lucide)
   - onSelect: for now, show a toast (sonner) with "Executed: {command.name}" — actual execution deferred since it requires active session context
   - Only show group when search starts with '/' OR when fuse matches command names

6. **ProjectGroup.tsx** (CMD-15):
   - Fetches projects from `GET /api/projects` via apiFetch on mount
   - Interface: `Project { name: string; path: string; isActive?: boolean }`
   - Each item: label = project.name, icon = FolderOpen (lucide), active project gets a dot indicator
   - onSelect: call `apiFetch('/api/projects/switch', { method: 'POST', body: JSON.stringify({ name: project.name }) })` then reload or navigate, then `onClose()`
   - If only one project, don't render this group

7. **RecentGroup.tsx** (CMD-14):
   - Props: `recents: RecentEntry[]`, `onClose: () => void`

   - onSelect: re-execute the action based on entry.id pattern (nav-* = navigation, session-* = session, action-* = action)
