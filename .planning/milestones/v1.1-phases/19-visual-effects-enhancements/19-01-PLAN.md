  <action>
Create `src/src/components/effects/` directory with three effect components, source-level cherry-picked from React Bits and adapted to OKLCH design tokens.

**SpotlightCard (SpotlightCard.tsx + SpotlightCard.css):**
- Wrapper component that tracks mouse position via `onMouseMove` and sets `--mouse-x`/`--mouse-y` CSS custom properties on the element
- CSS `::before` pseudo-element renders a `radial-gradient(circle at var(--mouse-x) var(--mouse-y), ...)` that fades in on hover
- Spotlight color: `oklch(from var(--accent-primary) l c h / 0.08)` -- soft glow using primary accent
- Props: `children: ReactNode`, `className?: string`
- Use `useRef<HTMLDivElement>` for the container, `getBoundingClientRect()` for mouse offset
- Transition: `opacity var(--duration-normal) var(--ease-out)`
- `@media (prefers-reduced-motion: reduce)`: `::before { display: none }`

**ShinyText (ShinyText.tsx + ShinyText.css):**
- CSS-only shimmer effect. NO Framer Motion (Constitution bans it).
- `linear-gradient(120deg, var(--text-muted) 40%, var(--accent-primary) 50%, var(--text-muted) 60%)` with `background-clip: text` and `-webkit-text-fill-color: transparent`
- `@keyframes shiny-sweep` animates `background-position` from `100%` to `-100%` over 3s linear infinite
- Props: `children: ReactNode`, `disabled?: boolean`, `className?: string`
- When `disabled`, render children without effect class
- `@media (prefers-reduced-motion: reduce)`: remove animation, unset text-fill-color, restore `color: var(--text-muted)`

**ElectricBorder (ElectricBorder.tsx + ElectricBorder.css):**
- Uses StarBorder CSS gradient pattern (NOT Canvas-based ElectricBorder from React Bits -- too heavy)
- Two inner `<div>` elements (gradient-top, gradient-bottom) with `radial-gradient(circle, var(--accent-primary), transparent 10%)` backgrounds
- `@keyframes` animate `translateX` for orbiting glow along top/bottom edges
- Props: `children: ReactNode`, `active?: boolean`, `className?: string`
- `data-active` attribute controls visibility via CSS `opacity: 0` -> `opacity: 0.5` transition
- Transition: `opacity var(--duration-normal) var(--ease-out)`
- `@media (prefers-reduced-motion: reduce)`: gradient elements `display: none`
- Container has `position: relative`, `overflow: hidden`, `border-radius: 0.75rem`, inner content gets `position: relative; z-index: 1` to sit above gradient elements

All components: named exports only, `cn()` for class merging, no hardcoded colors, no inline styles for colors/durations. Follow research code examples in 19-RESEARCH.md closely.
