**Create `src/src/hooks/useMessageSearch.ts`:**
- Custom hook returning: `{ query, setQuery, isOpen, open, close, toggle, filterMessages, highlightText }`

- `highlightText(text: string): ReactNode` -- splits text around matches, wraps matches in `<mark className="bg-primary/20 rounded-sm">`. Returns original text when no query.
- 150ms debounce on the filter (use `setTimeout` ref pattern) -- store both `query` (immediate, for input display) and `debouncedQuery` (for filtering). The `filterMessages` function uses `debouncedQuery`.
- Search scope per CONTEXT.md: user messages, assistant messages, thinking blocks. Does NOT search tool call inputs/outputs.

- `exportAsMarkdown(messages: Message[], sessionTitle: string): void`
  - Format: `# {sessionTitle}\n\n` then for each message:
    - User: `## User\n\n{content}\n\n`
    - Assistant: `## Assistant\n\n{content}\n\n` (content already has markdown)

- Internal `slugify(text)` -- lowercase, replace non-alphanumeric with hyphens, trim
