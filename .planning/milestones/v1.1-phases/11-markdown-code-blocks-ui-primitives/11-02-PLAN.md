1. Create `src/src/components/chat/view/CodeBlock.tsx`:

   Props: `{ language: string; code: string }`

   Structure:
   ```
   <div className="bg-code-surface rounded-lg overflow-hidden my-3" style={{ minHeight }}>
     <CodeBlockHeader language={language} code={code} />
     <div className={scrollClasses}>
       {highlighted content or plain fallback}
     </div>
   </div>
   ```

   **CodeBlockHeader** (internal component or inline):
   - Flex row with `justify-between items-center px-3 py-1.5 border-b border-border`
   - Language label: `<span className="text-xs text-muted font-mono">{language}</span>`
   - CopyButton: uses `navigator.clipboard.writeText(code)`, shows Check icon + "Copied!" for 2s via setTimeout, then reverts to Copy icon + "Copy". Uses Lucide `Copy` and `Check` icons.

   **Highlighting logic**:
   - `useState<string | null>(null)` for highlighted HTML
   - `useDeferredValue(html)` for smooth swap (CODE-06)
   - `useEffect` calls `highlightCode(language, code)` from shiki-highlighter.ts, sets result. Cleanup with cancelled flag.
   - Fallback: `<pre className="whitespace-pre"><code>{code}</code></pre>`
   - Highlighted: `<div dangerouslySetInnerHTML={{ __html: deferredHtml }} />`

   **Line numbers** (CODE-03):
   - Only for blocks > 3 lines
   - CSS counter approach: wrap each line in a `<span>` with `display: block` and use CSS `counter-increment` + `::before` pseudo-element. OR: render a line number gutter column next to the code.
   - Simpler approach: When rendering the fallback (or wrapping Shiki output), add a left gutter with line numbers using a flex layout. The Shiki HTML output has `<span class="line">` elements that can be targeted via CSS counter.

   **Max height** (CODE-04):
   - `lineCount > 20 ? 'max-h-[400px] overflow-y-auto' : ''`

   **Min height for CLS prevention** (CODE-05):
   - `const estimatedHeight = Math.min(lineCount * 20, 400)` (20px per line estimate)
   - Set as inline `style={{ minHeight: estimatedHeight + 'px' }}`

   **Horizontal scroll** (CODE-08):
   - `overflow-x-auto` on the code container, `whitespace-pre` (no wrap)

   **Line number CSS**: Add to the component or a co-located CSS file:
   ```css
   .shiki code { counter-reset: line; }
   .shiki .line { counter-increment: line; }
   .shiki .line::before {
     content: counter(line);
     display: inline-block;
     width: 2rem;
     margin-right: 1rem;
     text-align: right;
     color: var(--text-muted);
     opacity: 0.5;
   }
   ```
