  <action>
1. Create `src/src/components/chat/view/MarkdownRenderer.tsx`:

   ```typescript
   import Markdown from 'react-markdown'
   import remarkGfm from 'remark-gfm'
   import rehypeRaw from 'rehype-raw'
   import { CodeBlock } from './CodeBlock'
   import type { Components } from 'react-markdown'
   ```

   Props: `{ content: string }`

   Define `components` object with overrides (MD-03):

   **`pre`**: Strip the wrapper -- just pass children through. react-markdown wraps fenced blocks in `<pre><code>`. Since we route fenced code to CodeBlock (which has its own container), the `<pre>` wrapper is redundant.
   ```tsx
   pre: ({ children }) => <>{children}</>
   ```

   **`code`**: Detect fenced vs inline by checking for `language-*` className (react-markdown v10 pattern):
   - If `className` matches `/language-(\w+)/`: route to `<CodeBlock language={match[1]} code={String(children).trimEnd()} />`
   - Otherwise: render inline `<code className="bg-code-inline rounded-sm px-1.5 py-0.5 font-mono text-[0.85em]">`

   **`a`**: External vs internal link detection (MD-06):
   - External (starts with `http`): `target="_blank" rel="noopener noreferrer"`
   - Internal: no target attribute
   - Styling: `text-primary underline-offset-2 hover:underline`

   **`blockquote`** (MD-05):
   - `border-l-2 border-primary/30 pl-4 text-muted italic`

   **`table`** (MD-03):
   - Wrap in `<div className="overflow-x-auto my-3">`
   - Table: `min-w-full border-collapse text-sm`
   - Style `th` and `td` with padding and border-bottom

   **`th`**: `text-left px-3 py-2 border-b border-border font-medium text-foreground`
   **`td`**: `px-3 py-2 border-b border-border text-secondary-foreground`

   **`h1`-`h6`**: Add appropriate Tailwind size classes and margin:
   - h1: `text-2xl font-bold mt-6 mb-3`
   - h2: `text-xl font-bold mt-5 mb-2`
   - h3: `text-lg font-semibold mt-4 mb-2`
   - h4-h6: `text-base font-semibold mt-3 mb-1`

   **`ul`**: `list-disc pl-6 my-2 space-y-1`
   **`ol`**: `list-decimal pl-6 my-2 space-y-1`
   **`hr`**: `border-border my-4`
   **`p`**: `my-2 leading-relaxed` (paragraph spacing)

   Render:
   ```tsx
   <div className="markdown-body text-foreground text-sm leading-relaxed">
     <Markdown remarkPlugins={[remarkGfm]} rehypePlugins={[rehypeRaw]} components={components}>
       {content}
     </Markdown>
   </div>
   ```

   Do NOT use `@tailwindcss/typography` prose classes. Style everything through component overrides.
