  <name>Task 1: useDraftPersistence hook</name>
  <files>src/src/components/chat/composer/useDraftPersistence.ts</files>
  <action>
Create `useDraftPersistence` hook managing text drafts per session:

**Storage:**
- `draftsRef = useRef(new Map<string, string>())` for fast in-memory access during session switches
- `localStorage` key: `'loom-composer-drafts'` for reload survival
- On mount: hydrate `draftsRef` from localStorage (parse JSON, construct Map). Wrap in try/catch for corrupt data.

**API:**
- `saveDraft(sessionId: string, text: string)`: If text is non-empty after trim, set in Map. If empty, delete from Map. Debounce localStorage write by 500ms (use `useRef` for timer).
- `loadDraft(sessionId: string): string`: Return Map.get(sessionId) or empty string.
- `clearDraft(sessionId: string)`: Delete from Map, sync to localStorage immediately (no debounce -- this is on send).
- `sessionsWithDrafts: Set<string>`: Computed from Map keys. Expose as a stable reference that updates when drafts change. Use `useState<Set<string>>` that gets updated whenever saveDraft/clearDraft modifies the Map.

**localStorage sync:**
- `syncToStorage()`: `localStorage.setItem(key, JSON.stringify(Object.fromEntries(draftsRef.current)))`
- Called debounced from saveDraft, immediately from clearDraft
- Cleanup: clear debounce timer on unmount

Named exports: `useDraftPersistence`
