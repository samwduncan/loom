## Cross-Phase Integration Issues

### Issue 1: Keyboard Shortcut Escape Guards Not Wired (CRITICAL)

**Affects:** LAY-05, TERM-15

`useTabKeyboardShortcuts.ts` and `useCommandPaletteShortcut.ts` check for `data-terminal` and `data-codemirror` ancestors to suppress keyboard shortcuts inside the terminal and editor. Neither attribute is placed on any DOM element:

- `TerminalPanel.tsx` root div: no `data-terminal`
- `TerminalView.tsx`: no `data-terminal`
- `CodeEditor.tsx` root div: no `data-codemirror`

**Impact:** Cmd+1-4 and Cmd+K will fire while typing in the terminal or code editor, interrupting user input.

**Fix:** Add `data-terminal=""` to TerminalPanel root and `data-codemirror=""` to CodeEditor root.

### Issue 2: DiffEditor Component Orphaned

**Affects:** ED-15, GIT-06

`DiffEditor.tsx` (82 lines) and `useFileDiff.ts` (127 lines) are built, tested, and functional. But `ChangesView.tsx` uses `useOpenInEditor` (plain file open) instead. No production code imports `DiffEditor` or `useFileDiff`.

**Fix:** Update `ChangesView.handleClickFile` to use `DiffEditor` (or add a `viewDiff` action to the file store).

### Issue 3: Open in Terminal Menu Item Missing

**Affects:** FT-09

`FileContextMenu` was built with 3 items. The 4th ("Open Containing Folder in Terminal") was deferred to Phase 25, but Phase 25 never added it.

**Fix:** Add "Open in Terminal" item that calls `setActiveTab('shell')`.
